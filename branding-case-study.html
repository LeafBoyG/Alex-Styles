Case Study: Building a Visual Identity for Castro Capital

We were tasked with creating a clear and consistent visual identity for a new organisation, Castro Capital.

The main user need was for the organisation's audience—partners, investors, and clients—to immediately understand its purpose and credibility. Previously, there was no single system, which could lead to inconsistent and untrustworthy communications.

Our goal was to build a simple, robust identity system that could be applied easily by their team across all materials, both print and digital.

How We Did It

We focused on creating a small number of reusable components and clear guidelines. This avoids ambiguity and ensures the identity is applied correctly every time.

The system is built on 3 core elements:

1. A Defined Colour Palette

We established a strict and simple palette.

  • Primary: A deep blue acts as the base for all communications. It provides a serious, stable foundation.
  • Accent: A single accent colour, orange, is used sparingly to draw attention to key information, such as a headline or a call to action.
  • Neutral: Black and off-white are used for text and background, ensuring high contrast and legibility.

By limiting the colour options, we make it easier for users to apply the identity correctly.

2. A Clear Typographic Hierarchy

Readability was the primary goal. We selected a single, robust sans-serif typeface for all headings. The weight and size are defined in a simple scale to create a clear hierarchy of information on the page. This helps users scan documents and find what they need quickly.

Body copy uses a standard, legible font to make long-form text easy to read.

3. A Consistent Layout

All documents are built on a simple grid structure. This provides consistency, whether the user is viewing a letterhead or a brochure. It ensures information is presented in a predictable way, which helps to build trust and understanding.

The Outcome

The result is a flexible but highly consistent identity system. It allows the Castro Capital team to produce new documents that are instantly recognisable and professional.

By focusing on clarity, consistency, and ease of use, we have provided a system that meets the organisation's needs and serves its users effectively.
